PROFILE

A LITTLE ABOUT ME

SOFTWARE ENGINEER

My journey to becoming a Software Developer all started with a google search into the fastest growing industries in the US today. I saw a lot of tech based fields but Software Development immediately caught my interest. I started taking lessons on FreeCodeCamp and Udemy and my love for code started to grow. Since then I have been building my skills in JavaScript, Ruby, Ruby on Rails, React / Redux, and with HTML and CSS. These technologies have allowed me to utilize my creativity and enthusiasm for building and problem solving. Currently I am working on a clone of a social app called Untappd. This app utilizes the Ruby on Rails framework and has been a really fun project where I have been able to incorporate my appreciation for craft beverages with all of the technologies I have been working hard to learn. Recently I added AWS in order to store beer photo uploads as well as User profile images, which lessens in app data storage. Also I have been working with a small group on an app that we built using MongoDB, Express, React / Redux and Node. This app allows you to store a wardrobe, and your favorite outfits based on that wardrobe. Engineering the React components so they would handle the appropriate boolean data, describing each outfit, was an interesting challenge and very rewarding to hash out.

As a full stack developer I still enjoy learning and challenging myself, other technologies I have implemented and learned besides the ones mentioned above are PostgresSQL, Github, Algorithms and Data Structures, SCSS, Webpack, RSpec, Canvas, Heroku, JBuilder, Capybara. As well as being a developer, I love to stay active whenever I can. I'm a huge hockey fan and play as often as I can. I also love soccer, volleyball, football and pretty much anything physically challenging.
